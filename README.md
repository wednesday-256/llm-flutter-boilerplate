## 🧠 AI Logic: Project "Woden"
Woden acts as a **Grounded Intelligence Layer**, specifically engineered to eliminate LLM hallucinations through a high-efficiency RAG (Retrieval-Augmented Generation) pipeline.

### **Key Implementations:**
*   **JSON-Augmented RAG:** User queries are dynamically cross-referenced against a structured JSON knowledge base containing Cycull's specific mechanics, stats, and lore.
*   **Contextual Injection:** A custom Dart-integrated logic layer identifies relevant data nodes and injects them into the system instructions to guarantee 100% factual accuracy.
*   **Chain-of-Thought (CoT) Prompting:** Specialized prompt engineering that mandates the model to validate its internal logic against the provided context before generating a final response.
